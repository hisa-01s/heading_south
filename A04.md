# ファイル
使用するファイルは以下である。
- 01_04.txt:本文作成の構成案、指示文

# 指示
- 01_04の構成の流れをもとに、00_05の本文の文言を参照し本文を執筆
- h2の下に、冒頭文を作成する
- 文章はPREP法に配慮し、一般人にも読みやすい文体とする
- 各h2,h3,h4の本文は450文字程度を目安に執筆
- 内容によって、表や箇条書き(リスト)を使用し、視覚的にわかりやすく編集した形で出力
- 太文字、strongタグ、**(アスタリスク)は、SEO上マイナスになる可能性があるため使用しない
- できあがった本文は01_05.txtのファイル名で保存
